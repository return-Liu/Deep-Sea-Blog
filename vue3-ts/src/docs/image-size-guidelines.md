# 📐 图片尺寸设计规范

## 🎨 随笔随记图片尺寸

### 推荐尺寸

- **主封面图片**: `1920 × 1080px` (16:9 比例)
- **缩略图**: `400 × 225px` (16:9 比例)
- **移动端优化**: `1200 × 675px` (16:9 比例)

### 设计特点

- ✅ 16:9 比例适合现代屏幕显示
- ✅ 1920px 宽度确保高分辨率屏幕清晰度
- ✅ 支持响应式设计，多设备适配
- ✅ 使用 `object-fit: cover` 保持比例
- ✅ 最小高度 300px，最大高度 500px

### 响应式适配

```css
/* 桌面端 */
height: 500px;
min-height: 400px;
max-height: 600px;

/* 平板端 */
height: 350px;
min-height: 300px;
max-height: 400px;

/* 手机端 */
height: 250px;
min-height: 200px;
max-height: 300px;
```

## 📸 摄影图片尺寸

### 推荐尺寸

- **主图显示**: `2048 × 1536px` (4:3 比例) 或 `1920 × 1280px` (3:2 比例)
- **高质量展示**: `2560 × 1920px` (4:3 比例) 或 `2880 × 1920px` (3:2 比例)
- **缩略图**: `600 × 450px` (4:3 比例) 或 `600 × 400px` (3:2 比例)

### 设计特点

- ✅ 4:3 和 3:2 是摄影常用比例
- ✅ 支持高分辨率显示，保持图片质量
- ✅ 使用 `object-fit: contain` 确保完整显示
- ✅ 支持多种摄影比例，不强制裁剪
- ✅ 最小高度 600px，最大高度 1000px

### 响应式适配

```css
/* 桌面端 */
height: 85vh;
min-height: 600px;
max-height: 1000px;

/* 平板端 */
height: 70vh;
min-height: 400px;
max-height: 600px;

/* 手机端 */
height: 60vh;
min-height: 300px;
max-height: 500px;
```

## 🚀 图片优化建议

### 文件格式

- **JPEG**: 适合照片，文件小，兼容性好
- **WebP**: 现代格式，压缩率高，质量好
- **PNG**: 适合透明背景，文件较大

### 压缩建议

- **随笔封面**: 质量 85-90%，文件大小 < 500KB
- **摄影图片**: 质量 90-95%，文件大小 < 2MB
- **缩略图**: 质量 80-85%，文件大小 < 100KB

### 性能优化

- ✅ 启用懒加载 (`loading="lazy"`)
- ✅ 使用 `image-rendering: crisp-edges` 提升清晰度
- ✅ 设置 `object-position: center` 防止变形
- ✅ 提供多种尺寸适配不同设备

### 上传限制

- **最大文件大小**: 5MB
- **支持格式**: JPG, PNG, WEBP, GIF
- **建议分辨率**: 最小 800px 宽度

## 🎯 最佳实践

1. **保持比例**: 始终使用 `aspect-ratio` 保持图片比例
2. **响应式**: 为不同设备提供合适的尺寸
3. **性能优先**: 使用适当的压缩和懒加载
4. **用户体验**: 确保图片加载流畅，无变形
5. **可访问性**: 提供 alt 文本和适当的对比度
