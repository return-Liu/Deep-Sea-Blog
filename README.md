# Deep Sea 管理端

## 项目简介

Deep Sea 管理端是一个基于 Vue3+ts+ElementPlus 的管理端，用于管理 Deep Sea 的后台数据。

## 项目启动

1. 安装依赖

```bash
npm install
# 或者使用 pnpm 更快的安装方式
npm install -g pnpm
pnpm install
```

2. 启动项目

```bash
npm run dev
pnpm run dev
```

3. 打包项目

```bash
npm run build
pnpm run build
```
