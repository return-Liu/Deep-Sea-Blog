# Deep Sea 管理端

## 项目简介

Deep Sea 管理端是一个基于 Vue3+ts+ElementPlus 的管理端，用于管理一刻时光的后台数据。

## 项目启动

1. 安装依赖

```bash
npm install
# 或者使用 pnpm 更快的安装方式
npm install -g pnpm
pnpm install
```

2. 启动项目

```bash
npm run dev
pnpm run dev
```

3. 打包项目

```bash
npm run build
pnpm run build
```

## 项目功能

1. 🔐 登录与认证系统
   📱 登录方式
   双登录方式：支持密码登录和邮箱验证码登录
   🤖 人机验证：集成极验验证码，防止机器人攻击
   ⏱️ 倒计时机制：防止频繁发送验证码，保护系统安全
   🔒 账户锁定：前端实现登录尝试限制，多次失败后临时锁定
   🔐 账户安全
   🔑 密码重置：支持通过邮箱重置密码
   🛡️ 密码加密：使用 md5 加密存储密码
   🧊 冻结处理：专门的账户冻结页面，展示冻结原因和解冻时间
2. 🏠 主页功能
   🎠 轮播展示
   💬 即时通讯聊天室：展示基于 Vue+Node.js 实现的聊天室功能
   📝 留言墙：展示一刻时光留言墙功能，用户可留下足迹
   🎨 Yike Design UI 框架：推广团队开发的前端 UI 框架
   🤖 智能助手
   🧠 AI 助手集成：集成科大讯飞星火深度推理模型 X1 大模型
   🖥️ 全屏聊天模式：支持全屏聊天界面，提升交互体验
   🔍 深度思考模式：提供更深入的 AI 推理能力（开发中）
   📊 数据统计
   👁️ 访问量统计：展示总访问量和今日访问量
   ➕ 内容创建快捷入口：快速创建博客文章、摄影作品、随笔随记
3. 📝 内容管理系统
   👤 个人分享
   📘 博客文章管理：创建、编辑、删除博客文章，支持标签分类
   📸 摄影作品展示：上传和管理摄影作品，支持 EXIF 信息展示
   📔 随笔随记记录：记录日常生活随笔，支持个性化展示
   🔍 内容详情查看与编辑：提供详细的内容查看和编辑界面
   💬 留言墙
   🎨 多样化留言背景：支持多种背景样式，个性化留言展示
   🗂️ 留言分类筛选：按不同分类查看留言内容
   👍 点赞和评论功能：用户可以对留言进行互动
   🚩 举报功能：维护留言墙环境的健康
   🖼️ 照片墙
   📁 图片分类管理：按不同分类管理照片
   📤 照片上传和编辑：支持照片上传和信息编辑
   🖼️ 图片浏览和分页：提供良好的图片浏览体验
4. 🔍 搜索功能
   🌐 全局搜索
   🔤 关键词搜索：支持通过关键词搜索内容
   🗂️ 按内容类型筛选：可筛选文章、摄影、随记等不同类型内容
   ✨ 搜索结果高亮：搜索关键词在结果中高亮显示
5. 👤 用户系统
   📄 个人资料
   🖼️ 头像上传和裁剪：支持头像上传和裁剪功能
   📝 个人信息编辑：编辑昵称、性别、地区、生日等个人信息
   ✍️ 个性签名设置：设置个人简介和签名
   🌈 昵称颜色自定义：支持自定义昵称显示颜色
   🔒 账户安全
   📊 安全等级评估：可视化展示账户安全等级
   📱 设备管理：查看和管理登录设备
   🔑 密码修改：支持修改账户密码
   🗑️ 账户注销：提供账户注销功能
   🔁 多账号管理
   🔄 同一特征码下多账号切换：支持在同一设备特征下切换不同账号
   📋 账号列表展示：清晰展示所有关联账号
6. ⚙️ 系统管理
   ❄️ 冻结管理
   🚫 账户冻结状态展示：清晰展示账户冻结状态
   📄 冻结原因和解冻时间显示：详细展示冻结原因和预计解冻时间
   ⏱️ 倒计时提醒功能：实时显示解冻倒计时
   📱 设备管理
   📱 登录设备列表查看：查看所有登录过的设备
   ℹ️ 设备详细信息展示：展示设备类型、登录时间、IP 属地等信息
   🚪 设备登出功能：支持远程登出特定设备
   📢 举报中心
   🚩 用户举报处理：处理用户举报内容
   💬 反馈管理：管理用户反馈信息
   📊 处理结果跟踪：跟踪举报和反馈的处理结果
7. 💬 互动功能
   💬 评论系统
   🗩 多层级评论结构：支持嵌套评论，形成讨论串
   😀 表情包支持：评论中可插入表情符号
   🎨 评论背景卡：支持个性化评论背景
   👍 点赞功能：对评论进行点赞互动
   🚩 举报功能：维护评论区环境健康
   🤖 智能聊天
   🗣️ 与 AI 助手深度对话：与 AI 进行自然语言对话
   ⌨️ 打字机效果回复：模拟真实打字效果
   🔍 深度思考模式：提供更深入的 AI 推理能力（开发中）
8. 📁 资源管理
   💻 本地文件
   📅 按年月分类文件浏览：文件按时间分类管理
   ℹ️ 文件详情查看：查看文件详细信息
   🖼️ 图片预览功能：支持图片在线预览
9. 📢 系统通知
   📣 系统更新
   🔔 系统通知查看：查看系统发布的通知
   📄 更新内容展示：展示系统更新详情
   🔄 账号切换提醒：提醒用户账号切换功能
10. ⚙️ 设置功能
    🎨 个性化设置
    🌓 主题模式切换：支持日间/夜间模式切换
    🌐 多语言支持：支持中文、英文等多种语言
    📐 自定义布局：支持界面布局自定义
    🔐 安全设置
    🔑 密码修改：支持修改账户密码
    🗑️ 账户注销：提供账户注销功能
    📱 设备管理：管理登录设备
    🛠️ 技术特点
    🚀 现代化框架：基于 Vue3 + TypeScript + Element Plus 开发
    📱 响应式设计：适配不同屏幕尺寸，提供良好的移动端体验
    🧩 组件化架构：功能模块化，便于维护和扩展
    😊 用户体验优化：加载状态、错误处理、友好提示
    🛡️ 安全性：验证码、密码强度验证、账户冻结机制
    这套系统提供了完整的博客平台功能，从前端展示到后台管理，从内容创作到社交互动，形成了一个功能丰富的生态系统。系统注重用户体验，提供了丰富的个性化设置，同时保障了账户安全和内容质量。
